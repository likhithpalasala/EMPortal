"Instructions to run the application"

Requirements:
1. Python3 latest version installed on your PC.
2. Flask modules installed using pip or any alternative methods.
3. Xampp installed on your PC.
4. Place the FlaskCRUDApp directory in the location "xampp/htdocs/ "


"How to run the Code base"
1. Open the xampp server and run Apache and MySql
2. Beside MySql, click on the Admin option and it redirects you to the phpMyAdmin page.
3. Open terminal and go to the directory where the FlaskCRUDApp is on your PC.
4.Now run the script "python app.py"
5. It generates the localhost link and open the link in the web browser.
